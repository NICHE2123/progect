JavaScript

JavaScript — JavaScript – мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили.

Когда появился C#?

Брендану Эйху, нанятым в компанию Netscape 4 апреля 1995 года, была поставлена задача внедрить язык программирования Scheme или что-то похожее в браузер Netscape. Первоначально язык назывался Mocha, затем его переименовали в LiveScript, а 4 декабря 1995 года LiveScript переименовали уже в привычное название JavaScript

Где используют?

Этот язык программирования, разработчики используют для создания интерактивных веб-страниц. Функции JavaScript улучшают взаимодействия пользователя с веб-сайтом
